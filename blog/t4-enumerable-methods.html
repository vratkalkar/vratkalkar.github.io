Enumerable Methods in Ruby

November 2nd, 2014

There are additional methods in Ruby that can be called on a object. These are referred to as Enumerables. There are countless enumerable methods that can be called on an object, with the full list viewable here.

The method I am going to focus on today is the #map method. The #map method can be called on an object and returns a new array with the results of running the passed code block on every element

This might sound similar to the #each method, however there is one significant difference. The #each method iterates through each element of an array, however #map returns a new modified array, while #each returns the original array.

Here are some quick facts about the #map method for reference:

  • The returned array is always the same size as the original enumerable
  • The returned array elements are the result of calling the passed code block on each element
  • You can also pass certain symbols as arguments to a block ( ex. week.map(&:reverse)) which returns the array string elements reversed.
